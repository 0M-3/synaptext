# SynapText: A Full stack app made as to view documents as a Synapse of Documents

## Frontend

Stack:

- Vite: React framework using Typescript.

## Backend

Stack:

- FastAPI: Python framework.
- Langchain: Python AI Agent framework.
- SQLAlchemy: Python Database framework.

## Running instructions

1. Run backend using Makefile

```sh
make build
make run
```

2. Run frontend using

```sh
make build-frontend
make run-frontend
```
